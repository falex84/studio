# **App Name**: AlexPC Genius

## Core Features:

- Hardware Catalog Display: Displays a real-time stock catalog of available hardware components with prices.
- AI-Powered Diagnostics: Allows users to submit support tickets and receive preliminary diagnostics for hardware issues via an AI tool. The LLM considers user-provided data when providing the description.
- Automated Ticket Submission: Facilitates the submission of support tickets to a WhatsApp number with a summary of the user's issue and preliminary AI diagnosis.
- Real-Time Currency Conversion: Converts USD prices to Bolivares using the official BCV exchange rate.
- Shopping Cart Functionality: Enables users to add products to a cart and calculate the total cost in both USD and Bolivares.
- Checkout Process: Guides users through the checkout process, offering multiple payment methods, and facilitates order placement via WhatsApp.
- PDF Report Generation: Allows users to generate a PDF report of their diagnostic report.

## Style Guidelines:

- Primary color: A vibrant, bold blue (#3863C0) that represents technology and trust.
- Background color: A desaturated, dark blue (#0A0B0F) provides a high-tech feel.
- Accent color: A brighter blue (#5581DE), analogous to the primary color, to highlight key UI elements.
- Font: Use 'Inter' (sans-serif) for both body and headlines for a clean, modern look.
- Use 'Lucide Icons' for a consistent, minimalist aesthetic throughout the app.
- Employ a grid-based layout to ensure responsiveness and consistency across different screen sizes.
- Incorporate subtle transitions and animations to improve user experience (e.g., when adding items to cart or submitting tickets).
